# SnapDigest Landing Page

## What This Is
A gorgeous, modern landing page for SnapDigest — an AI-powered news reader iOS app.

## Stack
- Next.js 15 + React 19 + TypeScript
- Tailwind CSS 4
- Static export (next export → GitHub Pages)

## Key Pages
- `/` — Main landing page
- `/privacy` — Privacy policy
- `/terms` — Terms of service

## App Features to Showcase
- AI-powered news curation (GPT-4.1-mini/nano)
- Story Mode — swipeable news digests like TikTok for news
- 5-minute audio briefings (TTS)
- Smart personalization — learns what you care about
- Breaking news alerts
- Multi-language (EN/KO/ES)
- Save & bookmark articles
- Share cards — beautiful social sharing
- Offline reading
- Dark mode

## Design Direction
Premium, modern, editorial. Apple-level polish. Think Artifact/Flipboard/Apple News quality.
Use 21st.dev-inspired components — modern glassmorphism, smooth animations, beautiful gradients.
Mobile-first responsive. Dark theme preferred for the hero section.
