# SnapDigest Landing Page

Marketing website for SnapDigest - the AI-powered daily news briefing app.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

This site is configured for static export (`output: 'export'` in next.config.js), making it perfect for Vercel deployment.

1. Push to GitHub
2. Connect repo to Vercel
3. Deploy automatically on every push

## Pages

- `/` - Landing page
- `/privacy` - Privacy Policy
- `/terms` - Terms of Service
